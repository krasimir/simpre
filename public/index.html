Welcome to SimPre
The Simplest Presentation Framework

  • Every <section> tag is a slide
  • Forward (➡️ and ⬇️), Backward (⬅️ and ⬆️)
  • Auto-scaling and centering
  • Syntax highlighting
  • Progress indicator

Download simpre.zip

The main reason for creating this tiny Framework is the lack of slide auto screen fit. I want to dump my content and somehow stop worrying about formatting. This is what SimPre does for me. Whether this is text, image, or a code snippet, it automatically scales and positions the slide, so it uses all the available space.

Next: basic usage ➡️

Basic usage

The first step is to download simpre.zip. Extract it. All the slides go to index.html.

To create a new slide just add a new <section> tag.

To make a slide with code don't use <pre> or <code> tags directly. Insert the following:
<script src="assets/is.js" data-file="slides/example.js"></script>
Where "slides/example.js" is a file that you create.

That's it. Happy presenting!
