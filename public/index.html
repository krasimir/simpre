Welcome to SimPre
The Simplest Presentation Framework

Press ➡️ to move forward.

The main reason for creating this tiny Framework is the lack of slide auto screen fit. I want to dump my content and somehow stop worrying about formatting. This is what SimPre does for me. Whether this is text, image, or a code snippet, it automatically scales and positions the slide, so it uses all the available space. Check out the next slide.
